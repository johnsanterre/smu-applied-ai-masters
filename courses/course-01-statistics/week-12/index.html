Probabilistic Graphical Models

This week introduces graphical models—visual representations of joint distributions that encode conditional independence. These enable efficient reasoning in complex probabilistic systems.

Learning Objectives

  • Construct Bayesian networks from conditional independence assumptions
  • Read d-separation to determine conditional independence
  • Understand Markov random fields and their applications
  • Factor joint distributions using graphical structure
  • Apply inference algorithms (variable elimination, belief propagation)

Key Concepts

  • Bayesian Network: DAG with nodes as variables, edges as dependencies
  • d-separation: Graphical criterion for conditional independence
  • Markov Random Field: Undirected graphical model
  • Factorization: P(X) = ∏ P(Xᵢ|Parents(Xᵢ))
