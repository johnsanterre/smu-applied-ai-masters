Hypothesis Testing

This week introduces hypothesis testing—the formal framework for making decisions based on data. We'll learn to test claims, understand error types, and interpret p-values correctly.

Learning Objectives

  • Formulate null and alternative hypotheses for research questions
  • Compute and interpret p-values correctly
  • Understand Type I and Type II errors and their tradeoffs
  • Apply common tests (z-test, t-test, chi-squared)
  • Recognize common misconceptions about hypothesis testing

Key Concepts

  • Null Hypothesis (H₀): Default claim to be tested
  • Alternative Hypothesis (H₁): Claim if H₀ is rejected
  • P-value: P(data as extreme | H₀ true)
  • Type I Error (α): Rejecting true H₀
  • Type II Error (β): Failing to reject false H₀
  • Statistical Power: 1 - β, probability of detecting true effect
