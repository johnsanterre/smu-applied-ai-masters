Week 8 Quiz: Hypothesis Testing

Test your understanding of hypothesis testing and p-values.

Question 1: A p-value of 0.03 means:

  • There is a 3% probability the null hypothesis is true
  • The effect is 97% likely to be important
  • If H₀ were true, theres a 3% chance of seeing data this extreme
  • The alternative hypothesis is proven correct

Question 2: A Type I error occurs when:

  • We fail to reject a true null hypothesis
  • We reject a true null hypothesis
  • We fail to reject a false null hypothesis
  • We correctly reject a false null hypothesis

Question 3: Statistical power is increased by:

  • Using a smaller sample size
  • Using a more stringent significance level (e.g., 0.01 instead of 0.05)
  • Reducing the effect size
  • Increasing sample size or effect size

Question 4: The multiple testing problem refers to:

  • Increased Type I error rate when running many tests
  • Decreased Type I error rate with more tests
  • Tests becoming more accurate with replication
  • The need for larger effects in medical studies

Question 5: If p = 0.12 and α = 0.05, we should conclude:

  • The null hypothesis is true
  • The alternative hypothesis is false
  • We fail to reject H₀; insufficient evidence against it
  • The test was conducted incorrectly
