Point & Interval Estimation

This week covers estimation theory—how to use sample data to estimate unknown population parameters. We'll learn about estimator properties and confidence intervals.

Learning Objectives

  • Define point estimators and their desirable properties (unbiasedness, consistency, efficiency)
  • Calculate bias, variance, and mean squared error of estimators
  • Construct and interpret confidence intervals
  • Understand the relationship between confidence level, sample size, and interval width
  • Apply t-distributions for small sample inference

Key Concepts

  • Estimator: A rule for computing an estimate from data
  • Bias: E[θ̂] - θ, systematic over/underestimation
  • MSE: Variance + Bias², overall estimation error
  • Confidence Interval: Interval with specified coverage probability
  • t-distribution: Used when σ is unknown and n is small
