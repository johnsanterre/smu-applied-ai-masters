Monte Carlo Methods

This week covers Monte Carlo methods—computational algorithms using random sampling for numerical computation. MCMC is essential for Bayesian inference in complex models.

Learning Objectives

  • Understand Monte Carlo integration and its variance properties
  • Implement importance sampling and rejection sampling
  • Apply Metropolis-Hastings algorithm
  • Understand Gibbs sampling as a special case
  • Diagnose MCMC convergence

Key Concepts

  • Monte Carlo Integration: E[f(X)] ≈ (1/n)Σf(xᵢ)
  • MCMC: Construct chain with target as stationary distribution
  • Metropolis-Hastings: Propose-accept/reject with acceptance ratio
  • Gibbs Sampling: Sample each variable from conditional
