Random Variables & Distributions

This week bridges abstract probability theory to numerical analysis by introducing random variables and their distributions. These concepts are essential for understanding how ML models represent and learn from uncertain data.

Learning Objectives

By the end of this week, you will be able to:

  • Define discrete and continuous random variables and explain their differences
  • Construct and interpret probability mass functions (PMF) for discrete variables
  • Construct and interpret probability density functions (PDF) for continuous variables
  • Use cumulative distribution functions (CDF) to compute probabilities
  • Identify and apply common distributions (Bernoulli, Binomial, Poisson, Uniform, Normal, Exponential)

Key Concepts

  • Random Variable: A function mapping outcomes to real numbers
  • PMF: Probability mass function for discrete random variables
  • PDF: Probability density function for continuous random variables
  • CDF: Cumulative distribution function F(x) = P(X ≤ x)
  • Support: The set of values where a distribution has non-zero probability

Topics Covered

  1. From sample spaces to numerical outcomes
  2. Discrete random variables and probability mass functions
  3. Continuous random variables and probability density functions
  4. Cumulative distribution functions and their properties
  5. Discrete distributions: Bernoulli, Binomial, Geometric, Poisson
  6. Continuous distributions: Uniform, Normal (Gaussian), Exponential
  7. Relationships between distributions
