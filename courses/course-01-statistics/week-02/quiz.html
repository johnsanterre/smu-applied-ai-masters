Week 2 Quiz: Random Variables & Distributions

Test your understanding of random variables, probability mass/density functions, and common distributions.

Question 1: What is the key difference between a PMF and a PDF?

  • PMF values are probabilities; PDF values must be integrated to get probabilities
  • For a PDF, P(X = x) = 0 for any specific value x; for a PMF, P(X = x) can be positive
  • PMF can be negative while PDF must be non-negative
  • PDF describes random variables; PMF describes deterministic outcomes

Question 2: Customer arrivals at a store follow a Poisson process. Which distribution models the time between consecutive arrivals?

  • Binomial
  • Normal
  • Exponential
  • Poisson

Question 3: If the CDF of X is F(x), and F(2) = 0.7, what is P(X > 2)?

  • 0.3
  • 0.7
  • 0.35
  • 0.5

Question 4: You receive 20 emails, and each independently has a 30% chance of being spam. The number of spam emails has which distribution?

  • Binomial(20, 20)
  • Binomial(0.3, 20)
  • Binomial(20, 0.7)
  • Binomial(20, 0.3)

Question 5: Which statement about probability density functions is TRUE?

  • f(x) can be negative for some values of x
  • f(x) ≥ 0 for all x, and ∫f(x)dx = 1
  • f(x) must always be less than or equal to 1
  • f(x) gives the probability that X equals x
