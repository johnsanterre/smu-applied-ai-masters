Python Basics & Environment Setup

This week introduces Python and establishes your development environment. You'll write your first programs and become comfortable with the tools used throughout this course.

Learning Objectives

  • Install Python and configure a development environment
  • Navigate the Python REPL and run Python scripts
  • Write and execute basic Python programs
  • Understand Python's philosophy and design principles
  • Use an IDE effectively (VS Code or PyCharm)

Key Concepts

  • Python Interpreter: Executes Python code line by line
  • REPL: Read-Eval-Print Loop for interactive coding
  • Script: A .py file containing Python code
  • IDE: Integrated Development Environment for writing code
