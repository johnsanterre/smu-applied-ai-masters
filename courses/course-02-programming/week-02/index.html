Variables, Types & Operators

This week covers Python's fundamental data types, variable assignment, and operators. Understanding these building blocks is essential for all programming.

Learning Objectives

  • Create and use variables with meaningful names
  • Work with Python's core data types (int, float, str, bool)
  • Apply arithmetic, comparison, and logical operators
  • Perform string operations and formatting
  • Convert between data types

Key Concepts

  • Variable: A named reference to a value in memory
  • Data Type: Classification determining valid operations
  • Operators: Symbols performing operations (+, -, *, /, ==)
  • Type Coercion: Converting between types with int(), str(), etc.
