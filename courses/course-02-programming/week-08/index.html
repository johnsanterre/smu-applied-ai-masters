Object-Oriented Programming

OOP enables organizing code around objects that combine data and behavior. This paradigm is central to Python's design and most ML frameworks.

Learning Objectives

  • Define classes with attributes and methods
  • Understand __init__ and instance vs class attributes
  • Implement inheritance and method overriding
  • Apply encapsulation principles
  • Use special methods (dunder methods)

Key Concepts

  • Class: Blueprint for creating objects
  • Object: Instance of a class
  • Inheritance: Child class inherits from parent
  • self: Reference to the current instance
