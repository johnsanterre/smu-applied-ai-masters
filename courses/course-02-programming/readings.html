Curated resources for mastering Python programming and computational thinking.

Primary Textbooks

Automate the Boring Stuff with Python (2nd Edition)

Author: Al Sweigart | Publisher: No Starch Press, 2020

Practical programming for total beginners. Free at automatetheboringstuff.com

Python Crash Course (3rd Edition)

Author: Eric Matthes | Publisher: No Starch Press, 2023

Fast-paced introduction with hands-on projects.

Supplementary Resources

Think Python (3rd Edition)

Author: Allen B. Downey

Computational thinking approach to learning Python.

Fluent Python (2nd Edition)

Author: Luciano Ramalho | Publisher: O'Reilly, 2022

Deep dive into Python's advanced features (for motivated students).

Python for Data Analysis (3rd Edition)

Author: Wes McKinney | Publisher: O'Reilly, 2022

Definitive guide to Pandas by its creator.
