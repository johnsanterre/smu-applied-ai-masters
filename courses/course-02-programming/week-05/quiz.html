Week 5 Quiz: Lists & Tuples

Q1: What is the main difference between lists and tuples?

  • Lists are mutable; tuples are immutable
  • Lists use (); tuples use []
  • Tuples are ordered; lists are not
  • Lists can only hold numbers

Q2: What is [\"a\", \"b\", \"c\", \"d\"][2]?

  • \"a\"
  • \"b\"
  • \"c\"
  • \"d\"

Q3: What is [10, 20, 30, 40][1:3]?

  • [10, 20, 30, 40]
  • [20, 30]
  • [20, 30, 40]
  • [30, 40]

Q4: Which method adds an element to the end of a list?

  • add()
  • insert()
  • push()
  • append()

Q5: What does [x**2 for x in range(5)] produce?

  • [0, 1, 4, 9, 16]
  • [0, 2, 4, 6, 8]
  • [0, 1, 2, 3, 4]
  • [1, 4, 9, 16, 25]
