Functions & Modular Design

Functions enable code reuse, abstraction, and modularity. This week covers function creation, parameters, return values, and scope.

Learning Objectives

  • Define functions with def and return statements
  • Use positional, keyword, and default parameters
  • Understand variable scope (local, global, nonlocal)
  • Write docstrings and self-documenting code
  • Apply DRY principle through function abstraction

Key Concepts

  • Function: Reusable block of code defined with def
  • Parameters: Inputs a function accepts
  • Return Value: Output a function produces
  • Scope: Where variables are accessible
