Week 6 Quiz: Dictionaries & Sets

Q1: What is the average time complexity of dictionary lookup?

  • O(n)
  • O(1)
  • O(log n)
  • O(n²)

Q2: Which can be used as a dictionary key?

  • [1, 2, 3] (list)
  • {\"a\": 1} (dict)
  • \"hello\" (string)
  • {1, 2, 3} (set)

Q3: What method returns None if a key doesnt exist?

  • dict.get(key)
  • dict[key]
  • dict.find(key)
  • dict.getitem(key)

Q4: What is set([1, 2, 2, 3, 3, 3])?

  • [1, 2, 2, 3, 3, 3]
  • {3, 2, 1}
  • {1, 2, 2, 3, 3, 3}
  • {1, 2, 3}

Q5: What is {1, 2, 3} & {2, 3, 4}?

  • {1, 2, 3, 4}
  • {2, 3}
  • {1}
  • {1, 4}
