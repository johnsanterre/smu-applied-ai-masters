Control Flow & Logic

This week introduces decision-making and repetition in programs. Control flow structures enable programs to respond to conditions and process data efficiently.

Learning Objectives

  • Write conditional statements with if, elif, and else
  • Apply boolean logic using and, or, not
  • Create loops with for and while
  • Use break and continue for loop control
  • Avoid infinite loops and off-by-one errors

Key Concepts

  • Conditional: if/elif/else for branching logic
  • For Loop: Iterate over sequences
  • While Loop: Repeat while condition is true
  • Boolean Logic: and, or, not operators
