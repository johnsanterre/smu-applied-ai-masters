NumPy & Scientific Computing

NumPy is the foundation of Python's scientific computing stack. This week covers array operations essential for ML.

Learning Objectives

  • Create and manipulate NumPy arrays
  • Apply vectorized operations (avoiding loops)
  • Use broadcasting for efficient computation
  • Perform basic linear algebra operations
  • Index and slice multidimensional arrays

Key Concepts

  • ndarray: N-dimensional array with fixed type
  • Vectorization: Operations on entire arrays at once
  • Broadcasting: Automatic shape matching for operations
  • ufuncs: Universal functions for element-wise operations
