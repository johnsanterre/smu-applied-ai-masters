Week 12 Quiz: NumPy

Q1: What is key difference between NumPy arrays and Python lists?

  • Lists are faster
  • NumPy arrays have fixed type, enabling vectorization
  • Lists are ordered
  • NumPy has no methods

Q2: Which creates an array [0, 1, 2, 3, 4]?

  • np.zeros(5)
  • np.ones(5)
  • np.arange(5)
  • np.random.rand(5)

Q3: What is broadcasting?

  • Automatic shape expansion for element-wise operations
  • Combining arrays end-to-end
  • Converting to different dimensions
  • Selecting specific elements

Q4: What operator performs matrix multiplication?

  • +
  • &
  • *
  • @

Q5: What does arr.reshape(3, 4) do?

  • Returns the shape
  • Returns array with new dimensions (3 rows, 4 cols)
  • Deletes elements
  • Transposes the array
