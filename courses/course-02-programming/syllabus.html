Course 2: Introduction to Programming and Computational Thinking

Course Description

This foundational course develops fluency in Python programming and computational thinking essential for AI development. Students learn to write clean, efficient code; implement fundamental data structures and algorithms; and apply software engineering best practices including version control, testing, and debugging.

Prerequisites

  • No prior programming experience required
  • Basic computer literacy
  • Logical thinking and problem-solving aptitude

Learning Outcomes

Upon completion, students will be able to:

  • Write clean, efficient, well-documented Python code
  • Apply computational thinking to decompose complex problems
  • Implement and analyze fundamental data structures and algorithms
  • Design modular, reusable software components using OOP principles
  • Use version control (Git) for collaborative development
  • Debug, test, and refactor code systematically
  • Process and analyze data using NumPy and Pandas

Weekly Schedule

Week Topic Key Skills
1 Python Basics & Environment Setup Installation, REPL, IDE setup
2 Variables, Types & Operators Data types, expressions, string manipulation
3 Control Flow & Logic if/else, loops, boolean logic
4 Functions & Modular Design Function definition, parameters, scope
5 Data Structures: Lists & Tuples Sequences, indexing, slicing
6 Dictionaries, Sets & Collections Hash tables, set operations
7 File I/O & Data Processing Reading/writing files, CSV, JSON
8 Object-Oriented Programming Classes, objects, inheritance
9 Error Handling & Debugging Exceptions, try/except, debugging tools
10 Testing & Code Quality Unit tests, pytest, code review
11 Algorithms & Big-O Analysis Sorting, searching, complexity
12 NumPy & Scientific Computing Arrays, vectorization, linear algebra
13 Pandas & Data Manipulation DataFrames, data cleaning, analysis
14 Version Control with Git Git, GitHub, branching, merging
15 Capstone: Data Pipeline Project Integration project

Assessment Structure

  • Weekly Coding Assignments (40%): Hands-on programming exercises
  • Quizzes (15%): Concept checks after each module
  • Midterm Project (15%): OOP-based application
  • Capstone Project (25%): End-to-end data pipeline
  • Participation (5%): Discussion contributions, code reviews

Grading Scale

A: 90-100% | B: 80-89% | C: 70-79% | D: 60-69% | F: Below 60%

Required Materials

  • Primary Text: "Automate the Boring Stuff with Python" by Al Sweigart (free online)
  • Reference: "Python Crash Course" by Eric Matthes
  • Software: Python 3.11+, VS Code or PyCharm, Git
