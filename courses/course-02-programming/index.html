Foundation Phase

Introduction to Programming and Computational Thinking

Develop fluency in Python programming, algorithmic thinking, data structures, and software engineering practices foundational to AI development.

Learning Objectives

  • Write clean, efficient Python code following best practices
  • Apply computational thinking to decompose complex problems
  • Implement fundamental data structures and algorithms
  • Utilize version control and collaborative development workflows
  • Debug, test, and refactor code systematically
  • Build modular, reusable software components
