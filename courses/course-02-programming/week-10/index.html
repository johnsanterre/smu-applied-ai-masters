Testing & Code Quality

Writing tests ensures code correctness and enables confident refactoring. This week covers unit testing and code quality practices.

Learning Objectives

  • Write unit tests with pytest
  • Use assertions and test fixtures
  • Apply test-driven development (TDD)
  • Measure code coverage
  • Follow PEP 8 style guidelines

Key Concepts

  • Unit Test: Tests a single function/method in isolation
  • pytest: Popular Python testing framework
  • Fixture: Setup code for tests
  • Coverage: Percentage of code executed by tests
