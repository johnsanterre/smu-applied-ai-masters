Week 14 Quiz: Git

Q1: Which command saves staged changes to history?

  • git add
  • git commit
  • git push
  • git save

Q2: How do you create a new branch?

  • git init branch-name
  • git clone branch-name
  • git branch branch-name
  • git checkout branch-name

Q3: What causes a merge conflict?

  • Same lines changed in different branches
  • Pushing to remote
  • Cloning a repository
  • Conflicts never happen

Q4: What uploads commits to GitHub?

  • git add
  • git commit
  • git status
  • git push

Q5: How do you get latest changes from remote?

  • git log
  • git pull
  • git fetch
  • git merge
