Week 13 Quiz: Pandas

Q1: DataFrame vs Series?

  • DataFrame is 2D; Series is 1D
  • Series is 2D; DataFrame is 1D
  • DataFrame has no labels
  • Series has no index

Q2: df.loc vs df.iloc?

  • iloc is for labels
  • loc is for integers
  • loc uses labels; iloc uses integers
  • They are identical

Q3: How do you fill missing values with 0?

  • df.dropna(0)
  • df.fillna(0)
  • df.isna(0)
  • df.replace(0)

Q4: What does df.groupby(\"category\").mean() do?

  • Sorts by category
  • Reads a file
  • Concatenates DataFrames
  • Computes mean for each category

Q5: Which combines DataFrames on common columns?

  • pd.merge()
  • pd.concat()
  • df.append()
  • df.combine()
