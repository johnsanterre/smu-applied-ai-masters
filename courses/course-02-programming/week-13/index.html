Pandas & Data Manipulation

Pandas provides powerful data manipulation tools. This week covers DataFrames for real-world data analysis.

Learning Objectives

  • Create and manipulate DataFrames
  • Select, filter, and sort data
  • Handle missing values
  • Group and aggregate data
  • Merge and join datasets

Key Concepts

  • DataFrame: 2D labeled data structure
  • Series: 1D labeled array
  • loc/iloc: Label vs integer-based indexing
  • groupby: Split-apply-combine pattern
