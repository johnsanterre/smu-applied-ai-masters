Week 7 Quiz: File I/O

Q1: Which mode opens a file for writing (overwriting)?

  • \"r\"
  • \"a\"
  • \"w\"
  • \"x\"

Q2: Why use \"with open()\" instead of just \"open()\"?

  • It automatically closes the file when done
  • Files never need to be closed
  • It prevents all file errors
  • It catches exceptions automatically

Q3: Which module is used to read/write CSV files?

  • json
  • csv
  • file
  • text

Q4: Which function parses a JSON string into Python objects?

  • json.parse()
  • json.read()
  • json.decode()
  • json.loads()

Q5: What is the modern way to handle file paths in Python?

  • String concatenation
  • os.path module only
  • pathlib.Path
  • filepath module
