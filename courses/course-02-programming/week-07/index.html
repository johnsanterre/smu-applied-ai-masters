File I/O & Data Processing

Programs must read and write data to files. This week covers text files, CSV, JSON, and data processing patterns.

Learning Objectives

  • Open, read, and write text files
  • Use context managers (with statement)
  • Parse and write CSV files
  • Work with JSON data
  • Handle file paths and directories

Key Concepts

  • Context Manager: with open() as f: ensures proper cleanup
  • File Modes: r (read), w (write), a (append)
  • CSV: Comma-separated values format
  • JSON: JavaScript Object Notation for data exchange
