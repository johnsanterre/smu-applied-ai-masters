Linear Regression & Regularization

Master linear models—the foundation of machine learning—and learn how regularization prevents overfitting.

Learning Objectives

  • Implement ordinary least squares regression
  • Interpret coefficients and R² score
  • Apply Ridge (L2) and Lasso (L1) regularization
  • Understand the bias-variance tradeoff
  • Use polynomial features for non-linear relationships

Key Concepts

  • OLS: ŷ = Xw, minimizes sum of squared errors
  • Ridge (L2): Adds λ||w||² penalty, shrinks coefficients
  • Lasso (L1): Adds λ||w||₁ penalty, enables feature selection
  • ElasticNet: Combines L1 and L2 penalties
