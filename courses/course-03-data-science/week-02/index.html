Data Preprocessing & Feature Engineering

Clean, transform, and engineer features that make models perform better.

Learning Objectives

  • Handle missing values appropriately
  • Encode categorical variables (one-hot, label, target)
  • Scale and normalize numerical features
  • Create new features from existing data
  • Use sklearn transformers and ColumnTransformer

Key Concepts

  • Imputation: Filling missing values (mean, median, mode)
  • One-Hot Encoding: Converting categories to binary columns
  • StandardScaler: Zero mean, unit variance
  • Feature Engineering: Creating predictive features
