Capstone: End-to-End ML Project

Apply everything you've learned in a complete machine learning project.

Project Requirements

  • Choose a dataset and define a prediction task
  • Perform exploratory data analysis
  • Build a preprocessing and feature engineering pipeline
  • Train and evaluate multiple models
  • Tune hyperparameters systematically
  • Deploy model as an API or notebook demo

Suggested Domains

  • Healthcare: Disease prediction, readmission risk
  • Finance: Credit scoring, fraud detection
  • Retail: Demand forecasting, churn prediction
  • Custom: Bring your own dataset
