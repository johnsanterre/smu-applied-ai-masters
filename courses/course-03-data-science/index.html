Foundation Phase

Applied Data Science and Machine Learning

Master the complete machine learning pipeline using scikit-learn and modern tools. Learn linear models, tree-based methods, clustering, dimensionality reduction, neural networks, and best practices for building production ML systems.

Learning Objectives

  • Build end-to-end ML pipelines with scikit-learn
  • Implement linear regression, logistic regression, and regularization
  • Apply tree-based models: decision trees, random forests, gradient boosting
  • Use clustering and dimensionality reduction techniques
  • Understand neural network fundamentals with PyTorch
  • Evaluate models and tune hyperparameters systematically
