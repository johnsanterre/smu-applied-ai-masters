Course 3: Applied Data Science and Machine Learning

Course Description

This hands-on course covers the complete machine learning pipeline using scikit-learn, pandas, and PyTorch. Students learn to preprocess data, build and evaluate models, tune hyperparameters, and deploy ML systems. Topics span supervised learning (regression, classification), unsupervised learning (clustering, dimensionality reduction), and neural network fundamentals.

Prerequisites

  • Course 1: Statistical Inference and Probabilistic Modeling
  • Course 2: Introduction to Programming and Computational Thinking

Learning Outcomes

  • Build production-quality ML pipelines with scikit-learn
  • Implement and evaluate regression and classification models
  • Apply ensemble methods for improved performance
  • Use clustering and dimensionality reduction appropriately
  • Build simple neural networks with PyTorch
  • Deploy models using modern MLOps practices

Weekly Schedule

Week Topic
1 Introduction to Machine Learning
2 Data Preprocessing & Feature Engineering
3 Linear Regression & Regularization
4 Logistic Regression & Classification
5 Model Evaluation & Cross-Validation
6 Decision Trees & Ensemble Methods
7 Random Forests & Gradient Boosting
8 Support Vector Machines
9 Clustering & K-Means
10 Dimensionality Reduction & PCA
11 Neural Network Fundamentals
12 Deep Learning with PyTorch
13 Hyperparameter Tuning & Pipelines
14 Model Deployment & MLOps Basics
15 Capstone: End-to-End ML Project

Assessment Structure

  • Kaggle-style Competitions (30%): Weekly prediction challenges
  • Lab Notebooks (25%): Hands-on implementation exercises
  • Quizzes (15%): Weekly concept checks
  • Capstone Project (25%): End-to-end ML pipeline
  • Participation (5%): Discussions and peer review

Required Materials

  • Primary Text: "Hands-On Machine Learning" (3rd Ed.) by Aurélien Géron
  • Software: Python 3.11+, scikit-learn, pandas, PyTorch
