Course Readings

Essential resources for Applied Data Science and Machine Learning.

Primary Textbook

Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow (3rd Edition)

Author: Aurélien Géron

The definitive practical guide to ML with Python. Covers all major algorithms with code examples.

Supplementary Resources

An Introduction to Statistical Learning (2nd Edition)

Authors: James, Witten, Hastie, Tibshirani

Accessible introduction to statistical learning methods. Free PDF available.

The Elements of Statistical Learning

Authors: Hastie, Tibshirani, Friedman

Comprehensive advanced reference for ML theory. Free PDF available.

Python Data Science Handbook

Author: Jake VanderPlas

Complete guide to NumPy, Pandas, Matplotlib, and scikit-learn. Free online.
