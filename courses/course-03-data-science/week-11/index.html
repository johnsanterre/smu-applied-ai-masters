Neural Network Fundamentals

Understand the building blocks of deep learning.

Learning Objectives

  • Build a simple neural network from scratch
  • Understand forward and backward propagation
  • Apply activation functions (ReLU, sigmoid, softmax)
  • Train with gradient descent and optimizers
  • Use MLPClassifier/MLPRegressor in sklearn

Key Concepts

  • Neuron: Weighted sum + activation function
  • Layer: Collection of neurons
  • Backpropagation: Chain rule for computing gradients
  • Optimizer: SGD, Adam for weight updates
