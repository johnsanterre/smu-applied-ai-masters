Logistic Regression & Classification

Predict categorical outcomes using logistic regression and understand classification fundamentals.

Learning Objectives

  • Implement logistic regression for binary classification
  • Understand the sigmoid function and probability interpretation
  • Extend to multi-class classification
  • Use decision boundaries and thresholds
  • Handle class imbalance

Key Concepts

  • Sigmoid: σ(z) = 1/(1+e⁻ᶻ) maps to [0,1]
  • Log Loss: Cross-entropy loss for classification
  • Multi-class: One-vs-rest or softmax
  • Threshold: Convert probability to class (default 0.5)
