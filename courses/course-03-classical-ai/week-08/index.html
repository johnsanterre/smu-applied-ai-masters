Week 8 of 15

Constraint Satisfaction Problems

Structure problems as variables with domains and constraints for efficient solving.

Learning Objectives

  • Formulate problems as CSPs
  • Apply backtracking search with heuristics
  • Implement constraint propagation (AC-3)
  • Use MRV and LCV heuristics
  • Recognize global constraints

Key Concepts

  • Variable: Unknown to be assigned a value
  • Domain: Possible values for a variable
  • Constraint: Restriction on variable combinations
  • Arc Consistency: Every value has consistent partner
