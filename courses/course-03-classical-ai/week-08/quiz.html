Week 8 Quiz: CSPs

Q1: CSPs differ from standard search by:

  • Using a goal test
  • Never using heuristics
  • Exploiting constraint structure
  • Having states and actions

Q2: MRV (Minimum Remaining Values) selects:

  • Variable with fewest legal values left
  • Value that rules out fewest choices
  • Arc with most constraints
  • Domain with most values

Q3: Least Constraining Value (LCV) heuristic:

  • Picks variable with most constraints
  • Chooses value that rules out fewest choices for neighbors
  • Returns to previous variable on failure
  • Removes inconsistent values from domains

Q4: Arc consistency (AC-3) ensures:

  • O(n) runtime
  • Domains remain unchanged
  • All constraints are satisfied
  • Every value has a consistent partner in each constraint

Q5: Which is a classic CSP?

  • Sudoku puzzle
  • Shortest path finding
  • Image classification
  • Linear regression
