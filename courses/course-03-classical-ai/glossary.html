Key Terms Glossary

Classical AI Glossary

Search Algorithms

State Space

The set of all possible states reachable from the initial state. Defines the "universe" in which the agent operates. Understanding state space size is crucial for predicting algorithm tractability.

Breadth-First Search (BFS)

Uninformed search that explores nodes level by level. Complete and optimal for unit-cost edges but requires O(b^d) memory. Foundation for understanding graph traversal.

Depth-First Search (DFS)

Uninformed search exploring deepest nodes first. Uses O(bd) memory but not complete in infinite spaces. Used in backtracking and constraint solving.

Heuristic

Function h(n) estimating cost from node n to goal. Good heuristics dramatically reduce search effort. Designing admissible heuristics is both art and science.

A* Search

Best-first search using f(n) = g(n) + h(n). Optimal when h is admissible and consistent. The most important informed search algorithm.

Admissible Heuristic

Heuristic that never overestimates true cost to goal. Required for A* optimality. Examples: Manhattan distance, straight-line distance.

Logic & Reasoning

Propositional Logic

Logic with Boolean variables and connectives (∧, ∨, ¬, →). Foundation for knowledge representation. Decidable but limited expressiveness.

First-Order Logic

Extends propositional logic with quantifiers (∀, ∃), predicates, and functions. More expressive but semi-decidable. Used in knowledge bases and theorem proving.

Inference

Process of deriving new sentences from existing knowledge. Methods include resolution, forward/backward chaining. Core of logical AI systems.

Satisfiability (SAT)

Determining if a logical formula can be made true. NP-complete but efficient solvers exist. Foundation for many verification and planning systems.

Constraint Satisfaction

CSP (Constraint Satisfaction Problem)

Problem defined by variables, domains, and constraints. Examples: Sudoku, map coloring, scheduling. Specialized algorithms exploit structure.

Arc Consistency

For every value in domain of X, there exists a consistent value in Y. Preprocessing technique that prunes search space. AC-3 is the standard algorithm.

Backtracking

Systematic search assigning variables one at a time. Returns when constraint violated. Enhanced with heuristics and propagation.

Planning & Agents

Intelligent Agent

Entity that perceives environment through sensors and acts through actuators. Central concept organizing AI research. Agents can be simple or complex.

STRIPS

Classical planning language with preconditions and effects. Actions transform states. Foundation of most planning research.

MDP (Markov Decision Process)

Sequential decision-making with probabilistic transitions and rewards. Framework for planning under uncertainty. Solved by value/policy iteration.
