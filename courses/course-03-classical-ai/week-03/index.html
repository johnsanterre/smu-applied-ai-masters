Uninformed Search Strategies

Explore search algorithms that use only problem definition, without domain-specific knowledge.

Learning Objectives

  • Implement BFS, DFS, uniform-cost search
  • Analyze time and space complexity
  • Compare completeness and optimality
  • Understand iterative deepening
  • Choose appropriate strategy for problem characteristics

Key Concepts

  • BFS: Explores level-by-level, complete, O(b^d) space
  • DFS: Explores depth-first, O(bd) space, not complete
  • UCS: Expands lowest-cost node first, optimal
  • IDS: Combines BFS completeness with DFS space
