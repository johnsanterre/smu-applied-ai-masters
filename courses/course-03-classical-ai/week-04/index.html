Informed Search & Heuristics

Harness domain knowledge through heuristics to guide search toward goals more efficiently.

Learning Objectives

  • Define heuristic functions and their properties
  • Implement greedy best-first search
  • Understand admissibility and consistency
  • Design heuristics from relaxed problems
  • Analyze heuristic quality and dominance

Key Concepts

  • h(n): Estimated cost from n to goal
  • Admissible: h(n) ≤ true cost (never overestimates)
  • Consistent: h(n) ≤ c(n,a,n') + h(n')
  • Dominance: h₁ dominates h₂ if h₁(n) ≥ h₂(n) for all n
