Week 4 Quiz: Heuristics

Q1: An admissible heuristic:

  • Overestimates the true cost
  • Always returns zero
  • Never overestimates the true cost
  • Equals zero at goal states

Q2: For the 8-puzzle, Manhattan distance is:

  • Sum of horizontal and vertical distances for each tile
  • Straight-line distance to goal
  • Number of misplaced tiles
  • Depth in the search tree

Q3: Greedy best-first search selects nodes by:

  • Path cost g(n)
  • Heuristic value h(n)
  • f(n) = g(n) + h(n)
  • Discovery order

Q4: Heuristic h₁ dominates h₂ if:

  • h₁ is admissible and h₂ is not
  • h₁(n) < h₂(n) for all n
  • h₁(n) is always 0
  • h₁(n) ≥ h₂(n) for all n

Q5: Heuristics are often derived from:

  • Relaxed versions of the problem
  • Solving the full problem first
  • Random guessing
  • Precomputed optimal solutions
