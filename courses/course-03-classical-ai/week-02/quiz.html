Week 2 Quiz: Problem Formulation

Q1: A well-defined search problem includes:

  • Initial state, actions, transition model, goal test, path cost
  • States, goals, sensors
  • Variables, domains, constraints
  • Percepts, actuators, environment

Q2: The 8-puzzle has approximately how many reachable states?

  • 9! = 362,880
  • 3^9 = 19,683
  • 9!/2 = 181,440
  • 81

Q3: What is the branching factor (b)?

  • Number of states in the problem
  • Average number of successor states
  • Cost to reach the goal
  • Depth of optimal solution

Q4: State spaces are typically represented as:

  • Binary trees
  • Undirected graphs
  • Hash tables
  • Directed graphs

Q5: What role does abstraction play in problem formulation?

  • Simplifies state space by ignoring irrelevant details
  • Adds complexity to the problem
  • Converts easy problems to hard ones
  • Makes problems impossible to solve
