Problem Formulation & State Spaces

Learn to transform real-world problems into formal search problems with states, actions, and goals.

Learning Objectives

  • Define state space, initial state, goal test, actions
  • Formulate problems as search problems
  • Analyze state space size and complexity
  • Identify path cost functions
  • Represent problems with graphs and trees

Key Concepts

  • State: Configuration of the problem at a point in time
  • Action: Transition from one state to another
  • Goal Test: Function determining if state is goal
  • Path Cost: Cumulative cost of actions taken
