Capstone: AI Agent Implementation

Build an intelligent agent that combines search, logic, and planning techniques.

Project Requirements

  • Implement a complete AI agent for a complex domain
  • Apply at least two classical AI techniques
  • Include search (informed or uninformed)
  • Demonstrate planning or CSP solving
  • Evaluate and compare approaches

Suggested Domains

  • Game AI: Chess, Go, or custom game
  • Puzzle Solver: Sudoku, crossword, scheduling
  • Planning Agent: Robot task planning
  • Knowledge System: Expert system with reasoning
