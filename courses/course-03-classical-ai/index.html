Classical Artificial Intelligence: Search and Logic

Explore the origins of AI through search algorithms, knowledge representation, logical reasoning, constraint satisfaction, and planning methods that laid the groundwork for modern intelligent systems.

Learning Objectives

  • Implement uninformed and informed search algorithms
  • Apply heuristic design principles to optimize search performance
  • Represent knowledge using propositional and first-order logic
  • Solve constraint satisfaction problems systematically
  • Design planning algorithms for goal-oriented agents
  • Evaluate trade-offs between different AI approaches
