Week 9 Quiz: Propositional Logic

Q1: P → Q is false only when:

  • P is false
  • P is true and Q is false
  • P and Q have same value
  • P is true and Q is true

Q2: KB ⊨ α means:

  • KB and α are the same
  • α is in KB
  • α is true in all models where KB is true
  • α can be derived from KB

Q3: Modus ponens states:

  • From P→Q and P, infer Q
  • From P→Q and ¬Q, infer ¬P
  • From P→Q and Q→R, infer P→R
  • From P∨Q and ¬P, infer Q

Q4: CNF (Conjunctive Normal Form) is:

  • Disjunction of conjunctions
  • Negations only on atoms
  • Standard form for implications
  • Conjunction of disjunctions (clauses)

Q5: A complete inference algorithm:

  • Only derives true conclusions
  • Can derive everything that is entailed
  • Always terminates
  • Has no contradictions
