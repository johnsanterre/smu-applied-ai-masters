Classical Planning

Generate action sequences to achieve goals using STRIPS-style planning.

Learning Objectives

  • Represent actions with STRIPS operators
  • Apply forward and backward state-space search
  • Use partial-order planning
  • Implement GraphPlan algorithm
  • Convert planning to SAT

Key Concepts

  • STRIPS: Actions with preconditions, add/delete effects
  • State-Space: Search through world states
  • Plan-Space: Search through partial plans
  • GraphPlan: Build planning graph, extract solution
